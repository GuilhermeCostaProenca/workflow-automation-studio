# PLAN - AutoTarefas

## Project north
Deliver a functional MVP that is demonstrable and portfolio-ready.

## Phase 1 - Foundation
1. Consolidate PRD and MVP scope.
2. Close architecture and final stack.
3. Prepare base repository structure.

## Phase 2 - Build
1. Implement core flow end-to-end.
2. Ensure business rules, validation and observability.
3. Cover critical path with tests.

## Phase 3 - Polish
1. Enforce CI quality gate.
2. Finalize setup and architecture docs.
3. Publish v1.0.0 with release notes.

## Definition of done
- Core flow validated.
- CI green.
- Documentation complete.
- Release published.
